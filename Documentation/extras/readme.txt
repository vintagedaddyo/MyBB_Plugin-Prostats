How to use from XML feed:

1. Go to the plugin settings and turn the option "Active XML feed" on!

2. Go to http://YOUR_FORUM/index.php?stats=xml and check if works (there is any XML feed) or not.

3. Now for this "xml2html.php" xml parser, open it via a text editor and set your forum link (and the output templates):

4. Upload the XML parser wherever you want to show the forum stats.
